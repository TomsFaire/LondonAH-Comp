# London All Hands — Bitfocus Companion Config

Quick setup:

```bash
bash setup.sh
```

The script installs Homebrew, Companion, and Google Slides Opener; walks through the module and config import; then prompts for show variables and pushes them into Companion automatically.

Full operator reference: **[GUIDE.md](GUIDE.md)**
